Guide & Deployment (short)

  1. Rename this file to index.html.
  2. Create a GitHub repo (public is fine). Commit this file to the repository root.
  3. Add an assets/ folder with your images and videos. Example assets used in this file:
    • assets/demo_thumb.jpg
    • assets/video_thumb1.jpg
    • assets/video_thumb2.jpg
    • assets/poster1.jpg
    • assets/poster2.jpg
    • assets/demo.mp4 (optional local video)
  4. Push to GitHub. Enable Pages: Repository → Settings → Pages → Choose branch `main` → Save.
  5. Visit https://<username>.github.io/<repo>/ for your live site.

Recommended edits

  • Change the initials box at the top-left to your logo or initials.
  • Swap video tags for YouTube embeds if you want to save bandwidth (paste the iframe where the lightbox shows local <video>).
  • Compress images (use TinyPNG / Squoosh) and encode heavy videos with H.264 to reduce load time.

Optional improvements I can make

  • CMS support (Netlify CMS, GitHub Actions to auto-build).
